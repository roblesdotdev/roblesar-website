### Personal Website

My personal website is built using Remix and TailwindCSS. It supports Markdown
for personal notes and YAML for project listings. This is still in the
development stages, but you can explore the preview.

## Features

- [x] Formatting & Linting
- [x] Git hooks
- [x] Supports markdown and yaml content.
- [x] Styled using TailwindCSS
- [x] Responsive design
- [x] Syntax highlighting for code blocks
- [x] Support for tags
- [x] SEO friendly
- [x] Cache headers
- [ ] About page
- [ ] Contact page
- [ ] Light and dark theme
