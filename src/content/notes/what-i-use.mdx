Finding the right stack is key to solving problems efficiently and effectively.
This note is a log of the tools and technologies I commonly use, along with some
resources of interest.

### Tech

- [Typescript](https://typescriptlang.org/) - Essential for building javascript
  projects.
- [React](https://react.dev) - The framework I feel most comfortable building
  websites with.
- [Remix](https://remix.run) - One of the best web frameworks today.
- [Astro](https://astro.build) - My favorite framework for static sites.
- [Vite](https://vitejs.dev) - Tool that makes web development insanely fast.
- [Prisma](https://prisma.io) - Modern ORM for databases.
- [Zod](https://zod.dev/) - Typescript-first schema validation.
- [Conform](https://conform.guide/) - Compatible with Zod and Remix form
  validation.
- [TailwindCSS](https://tailwindcss.com) - The fatest way to prototype designs.
- [Radix UI](https://www.radix-ui.com/) - Primitives for building accessible
  design systems.
- [Framer Motion](https://www.framer.com/motion/) - Best library for animations.
- [Hono](https://hono.dev) - Useful for creating backends on the fly.
- [Vitest](https://vitest.dev) - Vite-native unit testing framework.
- [Playwright](https://playwright.dev/) - End-to-end testing framework.
- [Eslint](https://eslint.org/) - Most used javascript linter.
- [Prettier](https://prettier.io/) - Opinionated code formatter.

### Services

- [Github](https://github.com/) - Where my open source repositories are hosted.
- [Cloudflare](https://www.cloudflare.com/developer-platform/products/) - The
  best platform for creating & prototyping services quickly.
- [Turso](https://turso.tech/) - Powerful production-ready sqlite platform.
- [Resend](https://resend.com/) - The best service for sending emails.

### Tools

- [SVGOMG](https://jakearchibald.github.io/svgomg/) - SVG optimizer.
- [Excalidraw](https://excalidraw.com/) - Virtual whiteboard for sketching.
- [Web Test](https://www.webpagetest.org/): - Tool to check website performance.
- [Keycdn](https://tools.keycdn.com/) - Various purpose tools for the web.
- [Open Graph](https://www.opengraph.xyz/) - Tool to improve SEO.
- [Tinify](https://tinypng.com/) - Tool to compress and optimize images.

### Environment

- [Arch Linux](https://archlinux.org): My operating system of choice for
  development.
- [VSCode](https://code.visualstudio.com/): My go-to code editor.
- [Neovim](https://neovim.io/): My secondary editor for hobby projects and
  learning new languages.
- [tmux](https://github.com/tmux/tmux/wiki) - Terminal multiplexer for running
  multiple sessions.
- [Alacritty](https://alacritty.org/) - A fast, GPU-accelerated terminal
  emulator.

### Conclusion

These are some of the tools and resources I use in my everyday life. I am
probably forgetting some of them and others are changing or I am adding new
ones.
